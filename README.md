This repository builds an unpackable chrome extension unpackable in chrome dev. 

Currently works on: **minesweeperonline.com**. 

Current hotkey usage: **S** for highlights, **A** for autosolve

Keep in consideration site policy on ip/account ban upon autosolve detection

## Installation
- **Use this repo as a template**, then **clone locally**. You can also download a zip of the repo and unzip it on your computer.
- **Open [the extensions page](chrome://extensions)** in your browser: `chrome://extensions`. 
- **Toggle the "developer mode"**. 
- Click the button **_load unpacked extension_**.
- In popup window **select this folder**, then **click _ok_**.
- A new extension called _Sweeper_ should now be on your browser.
